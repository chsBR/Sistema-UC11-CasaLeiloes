# Repositório de Sistema de Casa de Leilões

Este projeto consiste em um sistema de cadastro de produtos para leilão. 
A aplicação possui duas telas, a tela de cadastro permite cadastrar produtos com nome e valor, 
e da acesso a tela de consulta que faz a listagem dos produtos cadastrados em uma tabela com:
- ID, Nome, Valor e Status
Nesta tela pode-se realizar a venda do produto cadastrado.

O sistema encontra-se em desenvolvimento, com algumas funcionalidades ainda 
não implementadas, que serão concluídas em atividades futuras.

## Tecnologias Utilizadas

- Java
- MySQL
- Git
- GitHub
